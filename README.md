## Subjects Covered

### Semester 3 (s3-class)
- **Data Structures** - Implementation of various data structures and algorithms in C
- **Object Oriented Programming** - Java programming assignments and experiments

### Semester 4 (s4-class)
- **Operating System** - OS concepts implementation in C (scheduling, memory management, etc.)
- **SSL Lab** - Security and scripting lab with Python and Shell programs

### Semester 5 (s5-class)
- **Applied Cryptography** - Cryptographic algorithms and ciphers implementation in Java

## Technologies Used
- **C** - Data structures and operating system concepts
- **Java** - Object-oriented programming and cryptographic implementations
- **Python** - Security tools and utilities
- **Shell Scripting** - System administration and automation scripts

## Note
This repository contains educational assignments and lab work for learning purposes.
